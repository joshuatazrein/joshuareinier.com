Joshua Reinier: Web Design

AM

AM is a sound-poem mixing radio static together with chance-based processing which diplays a noise of text fragments. The reader is challenged to reconcile conflicting voices and perspectives in a chorus of the current age.

MixedMedia.js

MixedMedia.js is a library for Canvas to create realistic brush, pencil, and splatter textures. Settings are customizable for every level of detail from individual grains to brushstrokes to overall fill shading. Below are several emulations of Agnes Martin paintings using the library.

RiverBank

RiverBank is a task manager implementing a two-tiered view: the Bank for storing unscheduled tasks, and the River a calendar view. Drag tasks from the Bank to the River to schedule them and easily organize.

Blog

This blog implements a PHP script which scans the server for updates, then presents an organized overview of both notes and sources. Tooltip previews of entries are available on hover, as well as a history and catalog of backlinks.
